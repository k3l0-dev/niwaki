# Cookbook

Task-oriented recipes — each one goes **describe → plan → push → verify**,
and every code block runs as a test in the SDK's own suite.  The recipes
live in the documentation: **<https://k3l0-dev.github.io/niwaki/cookbook/index.html>**

- [A three-tier application tenant](https://k3l0-dev.github.io/niwaki/cookbook/three-tier-app.html)

- [Static paths — attaching an EPG to ports](https://k3l0-dev.github.io/niwaki/cookbook/static-paths.html)
- [A complete L3Out](https://k3l0-dev.github.io/niwaki/cookbook/l3out-basic.html)
- [Fabric baseline — NTP, DNS, syslog, BGP RR](https://k3l0-dev.github.io/niwaki/cookbook/fabric-baseline.html)
- [Registering fabric nodes](https://k3l0-dev.github.io/niwaki/cookbook/node-registration.html)
- [VLAN migration — a day-2 change](https://k3l0-dev.github.io/niwaki/cookbook/vlan-migration.html)
- [Auditing a fabric](https://k3l0-dev.github.io/niwaki/cookbook/fabric-audit.html)
- [Troubleshooting connection & auth](https://k3l0-dev.github.io/niwaki/cookbook/troubleshooting-connection.html)
- [Troubleshooting a push](https://k3l0-dev.github.io/niwaki/cookbook/troubleshooting.html)
- [Designs in CI — a GitOps pipeline](https://k3l0-dev.github.io/niwaki/cookbook/gitops-pipeline.html)
- [Migrating from cobra](https://k3l0-dev.github.io/niwaki/cookbook/migrate-from-cobra.html)
