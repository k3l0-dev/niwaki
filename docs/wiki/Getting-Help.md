# Getting help

- **Questions & ideas** — [GitHub Discussions](https://github.com/k3l0-dev/niwaki/discussions)
- **Bugs & feature requests** — [issues](https://github.com/k3l0-dev/niwaki/issues/new/choose)
- **Missing DSL vocabulary** — the *vocabulary request* issue template;
  the [coverage matrix](https://k3l0-dev.github.io/niwaki/reference/vocabulary/coverage.html)
  shows what is curated today
- **Security** — never in a public issue: see
  [SECURITY.md](https://github.com/k3l0-dev/niwaki/blob/main/SECURITY.md)
- **Contributing** — [CONTRIBUTING.md](https://github.com/k3l0-dev/niwaki/blob/main/CONTRIBUTING.md)
  (reference-patch model, executable documentation conventions)
